# KaamConnect - AI-Enabled Job Platform for Daily Wage Workers

An innovative web-based employment platform specifically designed for daily wage and semi-skilled workers, connecting them with verified employers through AI-powered matching.

## ğŸš€ Getting Started

### Installation

1. Extract the ZIP file to your desired location
2. Open `index.html` in a modern web browser

### Usage

**For Workers:**
1. Click "Register" and select "I'm a Worker"
2. Fill in your details (or use Voice Profile)
3. Browse jobs or let AI match you with suitable opportunities
4. Apply to jobs and get hired!

**For Employers:**
1. Click "Register" and select "I'm an Employer"
2. Create your company profile
3. Post job requirements
4. Review matched workers and hire

**For NGOs:**
1. Visit the NGO Portal
2. Login with NGO credentials
3. Verify worker and employer profiles
4. Monitor platform statistics
